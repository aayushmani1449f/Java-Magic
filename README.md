# Practice Problems - Java

This workspace contains day-wise Java practice programs.

## Structure

- `Day1/` - Day 1 questions (`Q_1_1` ...).
- `Day2/` - Day 2 questions (`Q_2_1` ...).
- `Day3/` - Day 3 questions (`Q_3_1` ...).

## How to run

From the workspace root:

```powershell
# compile everything
javac Day1\*.java Day2\*.java Day3\*.java

# run one program (example)
java Day1.Q_1_1
```

Note: Files are intentionally kept in the **default package** (no `package` line) to match beginner practice setups.
